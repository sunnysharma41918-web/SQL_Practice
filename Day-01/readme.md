# 📘 Day 1 - PostgreSQL Basics (DBMS, RDBMS, Tables, Data Types)

## 🔹 1. What is DBMS?

**DBMS (Database Management System)** is software used to store, manage, and retrieve data efficiently.

### 📌 Key Points:
- Organizes data
- Allows easy access and updates
- Provides security and control

### 🔥 Examples:
- MySQL
- PostgreSQL
- Oracle
- SQL Server

---

## 🔹 2. What is RDBMS?

**RDBMS (Relational Database Management System)** stores data in structured tables (rows and columns) and maintains relationships between them.

### 📌 Features:
- Data stored in tables
- Uses Primary Key and Foreign Key
- Supports SQL (Structured Query Language)

### 📊 Example Table:

| customer_id | name  | city  |
|------------|------|------|
| 1          | Sunny | Delhi |
| 2          | Aman  | Noida |

---

## 🔥 DBMS vs RDBMS

| Feature       | DBMS        | RDBMS         |
|--------------|------------|--------------|
| Structure     | Files       | Tables        |
| Relationships | ❌ No       | ✅ Yes        |
| Keys          | ❌ Not required | ✅ Required |
| Example       | Old systems | PostgreSQL    |

👉 **Interview Tip:**  
RDBMS is an advanced version of DBMS that stores data in structured tables with relationships.
